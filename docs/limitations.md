# 项目限制与边界

## 明确不做什么

### 1. 不做实盘交易
- 当前为研究演示工具
- 统计显著性不足（t=-1.30）
- 不构成任何投资建议

### 2. 不做投资建议
- 因子结果不构成任何投资建议
- 过往表现不代表未来收益
- 投资需谨慎，风险自担

### 3. 不做实时系统
- 新闻数据为历史批量获取
- 未实现实时流处理
- 延迟为日度级别

### 4. 不做风险中性化
- 无 Barra 风格模型
- 未控制行业/市值暴露
- 存在风格因子敞口

## 统计结论

| 指标 | 数值 | 评估 |
|:-----|:-----|:-----|
| IC 均值 | -0.08 | 弱负相关 |
| t 统计量 | -1.30 | 不显著 (\|t\|<2) |
| p 值 | 0.194 | 不显著 (p>0.05) |
| 年化换手率 | 3.8-6.3x | 中等 |
| 估计年化成本 | 200-350 bps | 显著侵蚀收益 |

**结论**: 当前结果不满足传统因子标准（通常要求 \|t\|>2，IR>0.5）。

## 使用边界

| 场景 | 是否适用 | 说明 |
|:-----|:---------|:-----|
| 面试演示 | ✅ | 展示量化研究流程 |
| 方法论学习 | ✅ | 理解因子验证框架 |
| 实盘交易 | ❌ | 需进一步优化与验证 |
| 资管产品 | ❌ | 不满足监管要求 |
| 投资建议 | ❌ | 非投资建议 |

## 数据限制

| 限制 | 说明 |
|:-----|:-----|
| 样本期 | 约 6 个月，未覆盖完整市场周期 |
| 覆盖度 | 新闻数据源覆盖度不均，小盘股数据稀疏 |
| 数据源 | Event Registry API，存在访问限制 |
| 历史长度 | 建议扩展至 24 个月以上 |

## 改进建议

| 优先级 | 改进措施 | 预期效果 |
|:-------|:---------|:---------|
| 高 | 扩展历史数据至 24 个月 | 提高统计可信度 |
| 高 | 实施流动性筛选(ADV>5000万) | 降低冲击成本 30% |
| 中 | 降低调仓频率至周度 | 降低换手率 60% |
| 中 | 引入机器学习情感模型 | 提升 IC 20-30% |
